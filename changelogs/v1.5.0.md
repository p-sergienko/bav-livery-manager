## Changelog v1.5.0

## Features:

- A new bulk livery management toolset for administrators

  - Load multiple livery folders simultaneously and edit their `manifest.json` files in bulk
  - Mass edit all standard MSFS fields: content type, title, manufacturer, creator, package version, game version requirements, and dependencies
  - Mass edit BAV Manager metadata: registration name, aircraft type, engine, year, category, developer, simulator, and resolution
  - Auto-fill from the aircraft database: scans each livery folder for a registration and populates manager fields automatically
  - Copy assets to multiple livery folders simultaneously. Supports manager thumbnails (PNG), in-game thumbnails (PNG/JPG), and texture files (DDS, KTX2, cfg)
  - Automatically routes files to the correct subfolder based on FS20/FS24 package structure
  - Scans selected liveries for existing `texture.cfg` files and allows mass editing
  - Add, edit, and remove records from database used by the auto-fill feature
  - Automates post-build steps for livery packages by generating `layout.json` across an entire workspace and zipping them up for distribution

> For livery support and requests, please raise a ticket via the [**BAV Support Ticket System**](https://support.bavirtual.co.uk/)
