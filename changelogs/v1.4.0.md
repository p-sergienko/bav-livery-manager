## Changelog v1.4.0

## Features:
- Disk Usage section added to Settings — see at a glance how much space your installed BAV liveries and packages take up, with a per-drive breakdown (other apps / BAV / free), per-simulator totals for MSFS 2020 and MSFS 2024, and groupings by Developer, Aircraft, Simulator, or "Largest items"
- "Open" button next to each Community Folder path opens it directly in File Explorer
- Settings sections (Simulator Paths, Appearance, Disk Usage, App Updates & Changelog) now behave as a single accordion — opening one closes the others so the active section gets the full vertical room
- Cleaner Settings scrolling — the header and footer stay pinned while the section list scrolls when content overflows

### Bug Fixes:
- Sessions now persist much longer between launches; the app no longer prompts you to re-authenticate every 10–12 hours
- Fixed inability to uninstall liveries from the Updates page after an update became available
