## Changelog v1.2.0

## Features:
- Packages page now shows additional addons, including SpeedCore folders and sound packs
- Liveries that require SpeedCore will now install the SpeedCore package
- BAVMS support now means that with one click from the briefing page, users will be directed to their booked livery within the manager
- Improved download progress component
- Categories renamed to Subsidiaries
- Search page renamed to Liveries
- API is reworked from the scratch for better performance and more features in the future
- New font for liveries and page titles
- Various UI improvements and optimizations

### Bug Fixes:
- Fixed the tour not showing pictures and videos
