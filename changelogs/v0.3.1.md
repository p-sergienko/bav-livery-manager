## Changelog v0.3.1
### Features:
- Install accurate British Airways fleet liveries for both simulators and resolutions
- Keep your fleet up to date with ease using built-in livery updates that notify you when new versions are available and let you download them with a single click
- PMDG livery support allowing liveries to be detected in Operations Centre. Config files are also added automatically
- Auto-detect simulator paths for both FS20 and FS24, or select a custom location
- Convenient filtering options to enable members to search for the right livery
- The BAV Livery Manager keeps itself on latest version by auto-updating itself

### New Features:
- Changelog added to track new features and bug fixes
- Auto-updater works more efficiently
- Check for update button added
- Other improvements

### Bug Fixes:
- Fixed various bugs and improved overall stability

## Roadmap:
- A cancellation button that allows users to cancel downloads
- Improve the filters on My Downloads page
- Detailed livery information pages featuring airframe data, a full changelog for each livery, and quick access to aviation resources including FR24, AirNavRadar, and Planespotters
- Integrated SpeedCore support, allowing required packages to be installed automatically alongside compatible liveries
